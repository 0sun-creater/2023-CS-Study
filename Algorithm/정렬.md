## 정렬
- 데이터를 특정한 기준에 따라서 순서대로 나열하는 것
- 프로그램에서 데이터를 가공할 때 정렬해서 사용하는 경우가 많아서 프로그램 작성 시 가장 많이 사용되는 알고리즘 중 하나
- 정렬 알고리즘은 이진 탐색의 전처리 과정이기도 함
